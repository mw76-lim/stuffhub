HTML/CSS 그래프 샘플 세트 (안정 버전)

JS 없이, 복잡한 CSS 변수/마스크 없이, 브라우저 호환성 위주로 작성한 예시입니다.
값은 대부분 style="height:120px", style="width:60%" 처럼 인라인으로 바로 수정 가능합니다.

1) Simple Bar

각 막대의 높이를 px 단위로 직접 지정하는 기본 막대 그래프입니다.

JanFebMarAprMayJun

2) Grouped Bar

각 분기별로 A/B 두 개의 막대를 묶어서 보여주는 형태입니다.

Q1Q2Q3

3) Stacked Bar (세로)

하나의 막대 안에 여러 구간(A/B/C)을 쌓아서 구성 비율을 표현합니다.

Day 1Day 2Day 3

4) Stacked Bar (가로)

가로 방향으로 100%를 나누어 구성 비율을 표현합니다.

ABCD

5) Line Chart (SVG)

뷰박스 0~100 기준으로 좌표만 조정해서 손쉽게 수정 가능한 라인 차트입니다.

6) Area Chart (SVG)

라인 아래 영역을 채운 형태의 에어리어 차트입니다.

7) Donut Progress

SVG 원 둘레를 이용해 % 진행률을 도넛 형태로 표현합니다. (여기서는 65%)

8) Bullet Chart

목표 대비 실적을 단일 바와 타깃 마커로 표현합니다. (목표 80%, 실적 60%)

0%50%100%

9) Heatmap

칸별 강도를 클래스 이름(heat-1 ~ heat-5)으로 지정하는 히트맵입니다.

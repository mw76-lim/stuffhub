Ronda Korea 관리자 가이드 (Admin Guide)

본 문서는 Ronda Korea 웹사이트의 콘텐츠 및 팝업, 제품, 뉴스 관리 등 전반적인 관리 시스템 사용 방법을 안내합니다.

1. 관리 페이지 구조 개요

관리 페이지는 총 8개의 페이지로 구성되어 있으며, 독립적으로 운영되는 관리 페이지(5개)와 서비스 페이지에 병합되어 운영되는 관리 페이지(3개)로 구분됩니다.

A. 독립적인 관리 페이지 (5개)

  • Contact Us — 사용자가 제출한 [온라인 문의 내용] 확인 및 관리
  • 개인정보처리방침 — 개인정보 처리방침 내용 등록 및 수정
  • 회사소개서 — 회사소개서 PDF 파일 업로드 및 관리 (미등록시 비노출)
  • 팝업등록 — 신규 팝업 등록 및 기간 설정 (메인페이지에서 노출)
  • 팝업공지 — 공지 자세히보기 페이지 등록 및 관리 (팝업등록에서 사용)

B. 서비스 페이지에 병합된 관리 페이지 (3개)

  • Products 페이지 — 분류별 제품 (제목,설명,이미지) 데이터 관리
  • 뉴스룸 페이지 — 게시글 (제목,내용,썸네일이미지) 등록 및 관리
  • FAQ 페이지 — 자주 묻는 질문/답변 등록 및 관리

2. 관리자 접근 방법

  • ① 서비스 페이지 하단에 있는 [ADMIN] 버튼 클릭
  • ② 로그인 창에서 관리자 계정 입력 (ron***/ron*****)
  • ③ 로그인 성공 시 하단 버튼이 다음과 같이 변경됨:

[LOGOUT] [Contact Us] [개인정보처리방침] [회사소개서] [팝업등록] [팝업공지]

  • ④ 각 버튼 클릭 시 해당 독립 관리 페이지로 이동하여 콘텐츠 수정 가능

3. 페이지별 콘텐츠 관리 방식

A. 독립 관리 페이지 (5개)

  • 등록 / 수정 / 삭제
  • 게시 상태 설정은 (기간설정 / 등록여부:최신)로 결정
  • 파일 업로드 기능 (이미지, PDF 등)
  • 노출기간 설정 (시작일 / 종료일)

B. 서비스 통합 관리 페이지 (3개)

  • Products 페이지 — 제품명, 제품설명, 제품이미지 관리
  • 뉴스룸 페이지 — 뉴스/보도자료 등록 및 관리
  • FAQ 페이지 — 질문/답변 등록 및 관리

4. 운영 권장 사항

  • 로그인 후 브라우저 새 창에서 다른 페이지를 열지 마세요. (세션 충돌 방지)
  • 수정 후 반드시 저장하고, 새로고침으로 결과를 확인하세요.
  • 시간/노출기간 설정 시 시간 단위(YYYY-MM-DD hh:mm)까지 확인하세요.
  • 회사소개서 업로드 시 가장 최근 파일로 자동 대체됩니다.
  • 비정상 접근 또는 오류 발생 시 IT 관리자에게 문의하세요.

5. 기술 참고

  • Frontend: HTML5, CSS3, JavaScript (JQuery)
  • Backend: PHP 7.x (Cafe24 환경)
  • Auth: 세션 기반 로그인 시스템
  • Upload Path: /uploads/admin/, /uploads/popup/
  • Security: SSL 암호화, IP 접근 제한 가능

요약

  • 총 8개의 관리 페이지로 구성 (5개 독립, 3개 통합)
  • 접근 경로: 하단 ADMIN 버튼 → 로그인 → 관리 메뉴 표시
  • 주요 기능: 팝업, 뉴스, FAQ, 제품, 문의, 회사소개서 등 콘텐츠 관리

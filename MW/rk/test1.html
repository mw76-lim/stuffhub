판교-대장동 1층 하이브리드(사무/거주) 공간 배치 가이드

전제: 사무/거주 50:50 · 거실=회의실 · 마스터존(방1)=거주+집중 사무
목적: 가구 · 전자기기 · 비품 · 소모품 배치 “기본 패키지”를 공간별로 정의 ( 도식도 )
거실 = 회의/응대 주방/식당 = 라운지+탕비 마스터존 = 생활+집중업무 좌측 침실 = 서브오피스/게스트 테라스 = 휴식/야외미팅 현관/다용도실 = 운영 허브 욕실 = 위생/관리 다락 = 보관+작업+제습

1) 거실 → 회의실 겸 공용 업무 공간 공용 · 회사 얼굴

가장 넓은 중앙 공간을 “회의/응대/공용업무”로 고정. 대형 디스플레이와 화상회의 장비가 핵심.

가구

  • 10~12인 회의 테이블 (2400~3000mm)
  • 회의용 의자 10~12EA (메쉬 등받이 권장)
  • 벽면 수납 캐비닛 / 슬림 선반
  • 이동형 화이트보드

전자기기 (MON/AV)

  • 75~86" 대형 모니터 (MON)
  • 전자칠판/인터랙티브 디스플레이(선택)
  • 사운드바 또는 PA/스피커(AV)
  • 무선 프레젠터 + HDMI/USB-C 허브
  • 화상회의 카메라 + 회의용 마이크 스피커
비품/소모품: A4용지, 포스트잇, 보드마커, 회의 노트/펜, 멀티탭/케이블정리, 티슈/물티슈

2) 주방·식당 → 라운지 & 탕비 공간 업무 완충

간단 미팅/휴식/식사를 동시에 커버. 소모품(커피·컵·키친류) 표준화가 만족도를 좌우.

가구

  • 4~6인 테이블
  • 바체어 2~3EA (아일랜드)
  • 컵/식기 수납장

전자기기 (AP)

  • 커피머신/커피메이커
  • 전자렌지
  • 정수기(렌탈 포함)
  • 일반 냉장고 + 음료 냉장고
  • 공기청정기
소모품: 원두/캡슐, 종이컵/머그컵, 설탕/시럽, 키친타월/행주, 물티슈, 분리수거 봉투

3) 마스터존(방1) → 거주 + 집중 사무 개인 · 집중

침실 기능을 유지하면서도 장시간 집중업무가 가능한 셋업. 수납/동선/조명이 중요.

가구

  • 퀸 또는 슈퍼싱글 침대 + 협탁
  • 옷장/시스템장(드레스룸 연계)
  • 업무용 데스크 1600mm 내외
  • 인체공학 의자
  • 책장/서랍장

전자기기 (NB/MON/선택 AP·OF)

  • 노트북 1대 (NB)
  • 27" 모니터 1~2대 (MON)
  • 스탠드 조명(작업등)
  • 공기청정기(선택, AP)
  • 개인 프린터(선택, OF)
비품: 충전 스테이션, 케이블 정리, 서류 트레이/파일함, 개인 소형 금고(선택)

4) 좌측 침실 A → 서브 오피스(팀원용) 작업실

2인 작업실 기준. 공유 프린터/공유 수납을 이 방에 모으면 동선이 정리됨.

가구

  • 데스크 1400~1600mm × 2EA
  • 사무용 의자 × 2EA
  • 공용 서랍장/수납장
  • 화이트보드(선택)

전자기기 (PC/NB/MON/OF)

  • PC 또는 노트북 2대
  • 24~27" 모니터 2~4대
  • 공유 프린터(추천, OF)
  • 공기청정기(선택, AP)
소모품: 토너/잉크, 복사용지, 클리어파일, 스테이플러/심, 라벨지(자산/보관)

5) 좌측 침실 B → 게스트룸 / 임시 업무실 유연 공간

출장/외부 협업자 숙소 겸 임시 프로젝트 룸. 필요 시 촬영/편집실로 전환 가능.

가구

  • 소파베드 또는 싱글 침대
  • 이동형 데스크/접이식 테이블
  • 옷걸이 스탠드
  • 2인 테이블(선택)

전자기기 (선택)

  • 보조 모니터 1대(선택, MON)
  • 공기청정기(선택, AP)
  • 촬영/편집 장비 보관함(선택)
비품: 예비 침구/수건, 멀티탭, 기본 어메니티(간단), 쓰레기봉투

6) 테라스 → 휴식/야외 미팅 포인트 외부

분리된 휴식/흡연/짧은 미팅 공간. 방수·내후성 가구 중심으로 구성.

가구

  • 야외 테이블 + 의자(2~4인)
  • 스탠딩 테이블(선택)
  • 수납형 벤치(선택)

비품

  • 재떨이(필요 시)
  • 외부용 조명(센서등/스탠드)
  • 간이 청소도구(빗자루 등)
운영 팁: “테라스 전용 보관함” 1개 두면 청소/비품이 실내로 유입되지 않아 깔끔합니다.

7) 현관 → 출입·정리·방문 응대의 시작 동선 · 첫인상

업무/거주가 섞이는 공간일수록 현관은 “수납·청결·소지품 회수” 3가지를 표준화하면 편해집니다.

가구/수납

  • 신발장(기본 옵션 없으면) 또는 보조 신발장
  • 신발정리대/확장 선반(자주 신는 신발 구역 분리)
  • 현관 벤치(신발 신기용) + 하부 수납형 추천
  • 슬리퍼 보관함(방문자/실내용 분리)
  • 전신거울(외출/미팅 전 체크)

비품

  • 우산꽂이 + 물받이 트레이(빗물 바닥 방지)
  • 열쇠/소지품 트레이(법인차 키, 출입카드, 카드지갑)
  • 현관 매트(외부/내부 2중 구성 권장)
  • 미니 청소도구(먼지 제거 브러시/롤클리너)
  • 현관 조명(센서등/무드등, 필요 시)
소모품: 신발 탈취제/탈취볼, 제습제(신발장), 방향제, 물티슈, 마스크 보관(선택)
운영 팁: “자주 쓰는 구역(1단)”과 “계절/예비 구역(상단)”을 분리하면, 출근/외출 시간이 확 줄어듭니다.

8) 다용도실 → 세탁·청소·소모품 운영 허브 관리 · 운영

이 공간을 잘 잡으면 집/오피스가 동시에 깔끔해집니다. “수납 + 걸이 + 분리”가 핵심.

가전/설치

  • 세탁기
  • 건조기
  • 세탁기/건조기 직렬 선반(또는 앵글) + 진동 방지 패드
  • 빨래건조대(접이식) 또는 천장형 건조대(선택)

수납/정리

  • 빨래바구니(2~3칸 분리형: 흰/색/수건 등)
  • 다용도 수납장(세제/휴지/예비 소모품 보관)
  • 청소도구 걸이/수납랙(빗자루·밀대·먼지떨이)
  • 세제 디스펜서/라벨링(세탁/건조/표백 구분)

비품/소모품(추천 추가)

  • 세탁세제/섬유유연제/표백제 + 얼룩제거제
  • 세탁망 세트(니트/속옷/양말 전용)
  • 먼지 제거 롤러/테이프
  • 걸레/청소포 + 스프레이형 세정제
  • 쓰레기/분리수거 봉투(여기서 ‘재고’ 관리)
운영 팁: “휴지/세제/봉투” 같은 대용량 소모품은 다용도실에 ‘1곳 집중’이 효율 최고입니다.
체크: 콘센트 위치(세탁기/건조기/제습기), 배수/환기, 문 열림 방향(직렬 설치 간섭) 확인.

9) 화장실(1/2) → 위생·정리·관리 루틴 공간 공용/개인

기본은 ‘청소가 쉬운 구조’, 옵션은 ‘수납을 늘리는 구조’. 1/2 각각 동일 원칙으로 셋업하세요.

가구/수납

  • 욕실장(거울장 포함 여부 확인)
  • 슬림 수납장(틈새장: 세정제/휴지/여분 수건)
  • 변기 옆 수납장(휴지·물티슈·탈취제)
  • 수건장/수건걸이(부족하면 추가 설치)

비품

  • 샤워커튼/샤워부스 물막이(필요 시: 물튀김/곰팡이 예방)
  • 욕실 매트(흡수형) + 미끄럼 방지 패드
  • 빨래 바구니(작은 사이즈: 속옷/수건 임시 보관)
  • 변기솔 + 브러시(타일/세면대 겸용 구분 추천)

소모품(추천 추가)

  • 욕실 세정제(변기/타일/유리/세면대) + 곰팡이 제거제
  • 배수구 클리너/배수구 거름망
  • 휴지/핸드워시
  • 탈취제/방향제
  • 여분 칫솔/치약(게스트 대비)
운영 팁: “청소도구 1세트는 욕실에 상시” 두면, 큰 청소가 아니라 ‘즉시 청소’로 바뀝니다.
권장 루틴: 배수구(주 1회) · 거울/유리(주 1~2회) · 곰팡이(월 1회)로만 돌려도 관리가 쉬워집니다.

10) 다락방 → 보관 + 작업 + 휴식(선택) 습도 · 수납

다락은 ‘제습이 곧 유지비 절감’입니다. 보관을 넘어 “가벼운 작업실/휴식”까지 설계 가능.

수납/가구

  • 수납장(큰 박스/리빙박스용) + 라벨링
  • 선반/랙(앵글 선반 강추: 하중/확장성 좋음)
  • 접이식 테이블(작업용)
  • 접이식 의자
  • 빈백/1인 소파(휴식용, 선택)
  • 매트/러그(바닥 차가우면)

환경/관리(필수)

  • 제습기(다락은 습기 많음) + 배수 방식 확인(물통/연속배수)
  • 습도계(권장: 목표 45~55%)
  • 방습제/방충제(보관품 종류에 따라)
  • 작업등/스탠드 조명(어두우면 필수)
운영 팁: “박스는 투명/라벨 통일 + 선반에 카테고리 존”으로 두면, 다락이 창고가 아니라 ‘자산 저장소’가 됩니다.
보관 추천 카테고리: 계절 의류/침구, 캠핑/레저, 예비 가전, 문서/아카이빙(방습 필수).
주의: 습기 민감(종이/가죽/전자기기)은 제습기 + 밀폐박스 조합 권장.

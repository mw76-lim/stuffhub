업무 프로그램 설치 가이드

대상: RK 임직원 / 신규 노트북 지급 후
목적: 필수 업무 프로그램(OneDrive, Outlook, Teams, WeChat, WEHAGO) 설치 및 초기 설정 안내

목차
1. OneDrive 설정
2. Outlook 설정
3. Microsoft Teams 설치 · 설정 가이드
4. WeChat 설치
5. WEHAGO 접속 · 기능 확인

1. OneDrive 설정

회사 파일 동기화 및 자동 백업을 위해 반드시 설정합니다.

  1. 작업표시줄에서 OneDrive(구름 아이콘)를 클릭합니다.
    • 아이콘이 보이지 않으면 시작 → “OneDrive” 검색 후 실행
  2. 회사 계정(회사 이메일)으로 로그인합니다.
  3. 동기화 폴더 위치를 확인합니다.
    • 권장 예시: C:\Users\사용자\OneDrive - Ronda Korea
  4. 백업(자동 저장) 설정을 확인합니다.
    • 바탕화면 / 문서 / 사진 백업이 켜져 있는지 확인
  5. 동기화 상태가 “최신 상태”인지 확인합니다.
참고 업무 파일은 가능하면 OneDrive 폴더에 저장하세요. (PC 분실/교체 시 복구가 쉽습니다)

2. Outlook 설정

회사 메일 계정을 추가하고 송수신을 확인합니다.

  1. Outlook을 실행합니다. (없으면 사내 안내에 따라 설치)
  2. 회사 계정(회사 이메일)으로 로그인합니다.
  3. 메일함 동기화를 기다립니다.
  4. 테스트 메일을 본인에게 1통 보내고 수신을 확인합니다.
  5. 필요 시 서명/표시 이름 등 기본 설정을 확인합니다.
주의 회사 이메일은 Windows 로그인 계정이 아닙니다. Outlook에서만 회사 계정을 사용합니다.

3. Microsoft Teams 설치 · 설정 가이드

메신저/회의/화상 통화를 위해 Teams를 설치하고 로그인합니다.

  1. Teams 설치 여부를 확인합니다.
    • 시작 → “Teams” 검색 → 실행
    • 없으면 사내 안내 링크 또는 Microsoft Store에서 설치
  2. 회사 계정으로 로그인합니다.
  3. 알림/마이크/카메라 권한을 허용합니다.
  4. 장치 설정을 확인합니다.
    • Teams 설정 → 장치 → 스피커/마이크/카메라 선택
  5. 테스트: 임의 채팅 1회 또는 테스트 회의로 장치 동작을 확인합니다.
팁 회의가 잦다면 Teams의 “자동 시작” 옵션을 켜두면 편합니다.

4. WeChat 설치

대외/해외 커뮤니케이션이 필요한 경우 설치합니다.

  1. WeChat PC 버전을 설치합니다. (사내 안내 링크/공식 설치 경로 사용)
  2. WeChat 실행 후 로그인합니다.
    • 모바일 WeChat이 있으면 QR 스캔 로그인 방식이 일반적입니다.
  3. 알림 설정을 확인합니다. (필요 시)
  4. 파일 수신/전송 테스트를 1회 진행합니다.
주의 외부에서 받은 파일은 저장 위치를 확인하고, 업무 파일은 OneDrive로 옮겨 보관하세요.

5. WEHAGO 접속 · 기능 확인

WEHAGO 로그인 및 주요 기능(전자결재/공지 등) 동작 여부를 확인합니다.

  1. 브라우저(Chrome/Edge)를 실행합니다.
  2. WEHAGO 접속 주소로 이동합니다. (사내 안내 URL 사용)
  3. 회사 계정으로 로그인합니다.
  4. 주요 메뉴 동작을 확인합니다.
    • 공지/게시판 확인
    • 전자결재(또는 결재함) 접근 확인
    • 업무/메신저 등 사용 기능 접근 확인
  5. 즐겨찾기(북마크)에 WEHAGO를 추가합니다.
참고 접속 오류가 지속되면 네트워크 환경/권한 설정 이슈일 수 있으니 IT 담당자에게 문의하세요.
공통 중요 안내
• Windows 로그인 계정 ≠ 회사 계정
• 회사 계정은 Outlook / Teams / OneDrive / WEHAGO 등 업무 서비스 로그인에 사용
• 업무 파일은 반드시 OneDrive에 저장 권장 (백업/복구 용이)

RK 인사이트 세미나 아카이빙 운영 안내 (내부 공유)
세미나 종료 후 생성되는 강연 영상과 설문 결과를 기반으로, 사내 지식 자산을 축적하고 재활용하기 위한 아카이빙 운영 방식을 안내드립니다.
이번 안내의 핵심
• 회차별로 표준 폴더 구조와 기본 패키지를 만들어 저장합니다.
• 구성원은 내부 저장소에서 검색 → 시청/열람 → 재활용이 가능합니다.
• 자료는 RK 내부 교육·업무 참고용으로만 활용하며, 외부 공유는 금지합니다.
1) 아카이빙 목적 & 운영 원칙
목적
• 세미나 지식/사례를 누적하여 사내 학습 자산으로 전환
• 신규/전배 구성원의 온보딩 자료로 활용
• 팀별 반복 문의를 줄이고 업무 레퍼런스로 재사용

운영 원칙
• 자료는 내부 전용으로만 운영합니다(외부 공개/외부 공유 금지).
• 개인정보/민감정보 포함 시 비식별 처리 또는 공유 제한을 적용합니다.
• 표준 포맷을 적용해 검색 가능하고 재활용 가능한 형태로 저장합니다.
 
2) 회차별 저장 항목(기본 패키지)
기본 구성(필수)
• 01_Video : 최종 편집본(좌/우 합성) + 원본(선택)
• 02_Slides : 발표자료(PDF 권장) / 배포 가능 버전
• 03_Survey : 설문 결과 요약(핵심 지표 + 코멘트) + 원본 Export
• 04_Summary : 1페이지 요약(Summary.pdf) + 메타데이터(04_Metadata.txt)
• 05_Assets : 참고 링크/부록 자료(필요 시)
※ 파일명/폴더명은 아래 표준 규칙을 따릅니다.
 
3) 표준 폴더 구조 & 파일명 규칙
상위 폴더 예시
RK_Insight_Seminar / 2026 / 2026-03-25_RKInsight_06_박진범

회차 폴더 내부
01_Video / 02_Slides / 03_Survey / 04_Summary / 05_Assets

파일명 예시
• 2026-03-25_RKInsight_06_Final.mp4
• 2026-03-25_RKInsight_06_Slides.pdf
• 2026-03-25_RKInsight_06_Survey_Summary.pdf
• 2026-03-25_RKInsight_06_Summary.pdf
• 2026-03-25_RKInsight_06_Metadata.txt
 
4) 운영 흐름(수집 → 정리 → 검수 → 공유)
단계 내용
STEP A 자료 수집 영상(원본 2개) + 발표자료 + 설문 결과 Export를 확보합니다.
STEP B 편집/정리 좌/우 합성본 생성 → 불필요 구간 컷 → 음성/자막(필요 시) → 최종본 저장
STEP C 요약/메타데이터 Summary.pdf(1p)와 Metadata.txt를 작성해 검색/재활용이 가능하도록 표준화합니다.
STEP D 민감도 검수 민감정보 포함 여부를 점검하고 필요 시 블러/삭제/공유 제한을 적용합니다.
STEP E 내부 공유 내부 저장소 업로드 후, 공지 메일/게시판으로 링크와 요약을 공유합니다.
 
5) 구성원 활용 방법(검색·시청·재활용)
검색
• 폴더명/파일명(날짜·회차·강연자) + Metadata 키워드로 빠르게 찾습니다.

시청/열람
• Final.mp4로 전체 맥락 파악 → Summary.pdf로 핵심 복습 → Slides로 상세 확인

재활용
• 팀별 온보딩, 신규 과제 착수 시 레퍼런스, 유관부서 협업 시 공통 이해 기반으로 활용합니다.
 
6) 접근 권한 & 문의
• 아카이빙 자료는 RK 내부 구성원만 접근 가능합니다(외부 공유 금지).
• 접근 권한/링크 오류/자료 요청은 운영 담당자에게 문의해주세요.
